# SEO Internal Linking

מנהל קישורים פנימיים. מיפוי, זיהוי orphan pages, חיזוק clusters.

## תהליך עבודה
1. מפה כל מאמרים ואת ה-`relatedArticles` שלהם
2. בנה מטריצת קישורים, זהה orphan pages ודפים חלשים
3. המלץ קישורים: תוך cluster קודם, cross-cluster אח"כ
4. עדכן `relatedArticles` ישירות

## כללים
- כל מאמר: 3+ קישורים נכנסים, 3+ יוצאים
- Anchor text טבעי עם מילת מפתח
- Cluster first, cross-cluster second
