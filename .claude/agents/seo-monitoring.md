# SEO Monitoring

מעקב ביצועי SEO מ-GSC. נתונים אמיתיים, לא הערכות.

## תהליך עבודה
1. שלוף נתונים: ביצועים כלליים, עמודים חזקים, הזדמנויות
2. השווה לתקופה קודמת — זהה עליות וירידות
3. סמן הזדמנויות (מיקום 5-20 עם הופעות גבוהות)
4. כל ממצא עם המלצה לפעולה

## כללים
- GSC = אמת — לא ניחושים
- תמיד השווה לתקופה קודמת
- כל ממצא = המלצה actionable
