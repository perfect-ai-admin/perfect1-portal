## שלב 0 (מיידי): תיקון בעיות קריטיות — שבוע 1

### 0.1 פתרון בעיית הרנדור (קריטי)

**הבעיה:** SPA מבוסס React לא מרונדר על ידי גוגל כראוי.

**פתרונות (לפי סדר עדיפות):**

| פתרון | מורכבות | המלצה |
|--------|---------|--------|
| **Pre-rendering עם `prerender.io` / `rendertron`** | נמוכה | מומלץ לטווח קצר |
| **Vercel Prerender (ISR)** | בינונית | מומלץ לטווח בינוני |
| **מעבר ל-Next.js / Astro** | גבוהה | מומלץ לטווח ארוך |

**פעולה מומלצת מיידית:** הוספת `prerender-spa-plugin` ל-Vite build, או שימוש בשירות pre-rendering חיצוני כמו `prerender.io`.

### 0.2 עדכון Sitemap

ה-sitemap הנוכחי לא כולל את כל דפי הפורטל. צריך לעדכן אותו דינמית.

### 0.3 הגשה ל-Google Search Console

- אימות הדומיין ב-GSC
- הגשת sitemap
- בקשת אינדוקס לדפים העיקריים
- בדיקת Core Web Vitals

### 0.4 הוספת `canonical` tags

לכל דף פורטל צריך canonical tag שמצביע על ה-URL המלא.
